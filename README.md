# ⚡ scode — Your Terminal. Evolved.

Elite self-healing terminal AI agent.
100% Local | Multi-Provider | Immortal | Termux Ready

## One-Line Install
git clone https://github.com/sprizen-ai-system/scode.git
cd scode && pip install rich prompt-toolkit
ollama serve & python3 scode.py

## Features
- 100% Local (Ollama)
- Multi-Provider: OpenAI, Claude, Gemini, Groq
- File editing with diff preview
- Git integration
- Self-healing & self-evolving
- Works on Android (Termux)
